
POST /
Multipart formdata with parameters
  Parameters:
    - file => task file as plain tar file
    - notify => URL for notification when done
    - token => token to post to notification URL

Service:
  Runs the task in the sandbox and reads the output.
  
  Sends a POST request (Content-Type: application/x-www-form-urlencoded) to
  the notify URL with the following parameters:
    - token: the token given in the request
    - status: one of 'finished', 'failed', 'timeout'
    - output: the output.txt of the task
